# Igiwaz TV

Igiwaz TV is an open-source streaming app for Android TV.

The goal of this project is to build a modern TV interface inspired by Netflix with smooth navigation using a remote control.

## Features planned

- Android TV interface
- Netflix style layout
- Video player integration
- IPTV support
- Remote control navigation (DPAD)

## Technologies

- Java
- Android SDK
- Leanback Library
